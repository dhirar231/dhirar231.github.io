Configuration du Réseau

Plan d'Adressage IP

Le plan d'adressage utilise la technique VLSM (Variable Length Subnet Mask) pour une allocation efficace des adresses IP selon les besoins de chaque département.

Configuration des Routeurs
Routeur Interface Adresse IP Masque Réseau Area OSPF
RZ1 (Web) FastEthernet0/0 172.24.0.1 /18 (255.255.192.0) 172.24.0.0/18 Area 0
RZ2 (IT) FastEthernet0/0 172.24.64.1 /21 (255.255.248.0) 172.24.64.0/21 Area 0
RZ3 (DB) FastEthernet0/0 172.24.72.1 /23 (255.255.254.0) 172.24.72.0/23 Area 0
RZ4 (NFS) FastEthernet0/0 172.24.74.1 /24 (255.255.255.0) 172.24.74.0/24 Area 0
Note : Tous les routeurs participent à l'area 0 OSPF pour assurer une convergence rapide du routage.

Sécurité et NAT

Configuration NAT sur RZ1

Le routeur RZ1 est configuré pour fournir l'accès Internet à l'ensemble du réseau via NAT (Network Address Translation).

  • NAT Overload (PAT) configuré
  • ACL pour le trafic autorisé
  • Politiques de sécurité inter-VLAN
  • Filtrage par port sur les services sensibles
Attention : Les politiques de sécurité restreignent l'accès entre certains VLANs pour des raisons de sécurité.
